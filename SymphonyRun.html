Symphony Run

Symphony run is an endless runner type game. I made this project together with my project group in block 2 of year 2 of my CMGT studies at the hanzehogeschool Groningen. The game was made for a client that wanted to attract customers to classical music. the game features a unique and intuitive movement system where the player cal fly, dodge and run to avoid obstacles. My role for this project was a programmer.
