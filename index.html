Most Recent Project:

One of my most recent projects is called "SAM Simulator". In this project, as the name implies, the player plays as a SAM system operator! The SAM system the player is using is modelled afte rthe real-life MIM-23 SAM system. Im the game, everything is phyiscally modelled, the radar beam is a physical model, the radar screen accurately represents all the detected targets and their locations, and the missile has an actual flight model that is highly detailed. I am planning on finalizing and releasing this project later this year.

Build The
Nuke

Build the nuke is a personal project i started in the end of 2023. In the game, two teams of players spawn on an island to face off against eachother. they have to fight over and gather resources to win. Eventually one team will be able to build a nuke and nuke the other side and thus win the match. the game is made in UE5 and uses a dedicated server with replication for multiplayer.
